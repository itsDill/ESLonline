Tricky Sound Pairs üî§

These sounds are often confused by English learners. Practice each pair carefully!

S vs. Z (Voicing) Common
S = voiceless (air only) | Z = voiced (vibration)

Minimal Pairs:
‚Ä¢ sip / zip   ‚Ä¢ sue / zoo
‚Ä¢ rice / rise   ‚Ä¢ price / prize
‚Ä¢ loose / lose   ‚Ä¢ ice / eyes
‚Ä¢ bus / buzz   ‚Ä¢ peace / peas
üí° Put your hand on your throat - feel the vibration for Z!
L vs. R Tricky
L = tongue touches roof | R = tongue curls back

Minimal Pairs:
‚Ä¢ light / right   ‚Ä¢ lead / read
‚Ä¢ long / wrong   ‚Ä¢ alive / arrive
‚Ä¢ fly / fry   ‚Ä¢ glass / grass
‚Ä¢ collect / correct   ‚Ä¢ late / rate
üí° For L: tongue tip UP. For R: tongue tip DOWN and curled back!
TH /Œ∏/ vs. S/F Tricky
TH = tongue between teeth!

Minimal Pairs:
‚Ä¢ think / sink   ‚Ä¢ thick / sick
‚Ä¢ math / mass   ‚Ä¢ path / pass
‚Ä¢ three / free   ‚Ä¢ through / threw
‚Ä¢ thank / sank   ‚Ä¢ bath / bass
üí° Stick your tongue out slightly between your teeth for TH!
TH /√∞/ vs. D/Z Tricky
TH (voiced) = tongue between teeth + voice

Minimal Pairs:
‚Ä¢ this / dis   ‚Ä¢ that / dat
‚Ä¢ the / duh   ‚Ä¢ they / day
‚Ä¢ breathe / breed   ‚Ä¢ clothe / close
‚Ä¢ mother / mudder   ‚Ä¢ father / fodder
üí° Same tongue position as TH, but add voice vibration!
V vs. W Common
V = teeth on lip | W = lips rounded

Minimal Pairs:
‚Ä¢ vine / wine   ‚Ä¢ vet / wet
‚Ä¢ vest / west   ‚Ä¢ viper / wiper
‚Ä¢ very / wary   ‚Ä¢ vow / wow
‚Ä¢ vibes / wives   ‚Ä¢ veil / whale
üí° For V: bite your lower lip. For W: make a kissing shape!
B vs. V Common
B = both lips together | V = teeth on lip

Minimal Pairs:
‚Ä¢ bet / vet   ‚Ä¢ boat / vote
‚Ä¢ ban / van   ‚Ä¢ berry / very
‚Ä¢ base / vase   ‚Ä¢ best / vest
‚Ä¢ curb / curve   ‚Ä¢ bowel / vowel
üí° B = explosive with both lips. V = continuous with teeth!
P vs. B vs. F Basic
P/B = lips together | F = teeth on lip

Minimal Pairs:
‚Ä¢ pin / bin / fin   ‚Ä¢ pat / bat / fat
‚Ä¢ peel / beel / feel   ‚Ä¢ pack / back
‚Ä¢ cap / cab   ‚Ä¢ rip / rib
üí° P = voiceless puff. B = voiced. F = air through teeth!
SH vs. CH vs. J Common
SH = continuous | CH/J = stopped then released

Minimal Pairs:
‚Ä¢ ship / chip   ‚Ä¢ shop / chop
‚Ä¢ share / chair   ‚Ä¢ wash / watch
‚Ä¢ cheap / jeep   ‚Ä¢ choke / joke
‚Ä¢ match / Madge   ‚Ä¢ rich / ridge
üí° SH is like "shhhh". CH starts with T. J starts with D!

Length matters for meaning!

Minimal Pairs:
‚Ä¢ ship / sheep   ‚Ä¢ sit / seat
‚Ä¢ bit / beat   ‚Ä¢ fit / feet
‚Ä¢ full / fool   ‚Ä¢ pull / pool
‚Ä¢ luck / Luke   ‚Ä¢ cut / cute
üí° Long vowels: hold the sound longer and smile/round more!
N vs. NG Basic
N = tongue at front | NG = tongue at back

Minimal Pairs:
‚Ä¢ sin / sing   ‚Ä¢ thin / thing
‚Ä¢ ran / rang   ‚Ä¢ win / wing
‚Ä¢ ban / bang   ‚Ä¢ kin / king
‚Ä¢ ton / tongue   ‚Ä¢ run / rung
üí° For NG: push the back of your tongue UP. No "g" sound at end!
F vs. H Basic
F = teeth on lip | H = open throat

‚Ä¢ fat / hat   ‚Ä¢ fear / hear
‚Ä¢ food / hood   ‚Ä¢ fall / hall
‚Ä¢ fan / hand   ‚Ä¢ feet / heat
‚Ä¢ fair / hair   ‚Ä¢ fool / who'll
üí° F needs friction. H is just a breath of air!

Shadowing Practice üéØ

The Best Way to Improve Pronunciation:
  • Find a short video/audio with a native speaker (YouTube, podcasts)
  • Listen to one sentence at a time
  • Repeat IMMEDIATELY, copying rhythm, stress, and intonation
  • Record yourself and compare
  • Practice the same sentence until it sounds natural

üí° Tip: Focus on sounding natural, not perfect! Native speakers care more about rhythm and stress than individual sounds.
